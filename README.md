# Railly Studio Website

Официальный сайт небольшой инди-студии **Railly Studio**, созданный для представления команды, публикации информации о текущем проекте и поддержки разработки первой игры.

## 🌐 Демо

[🚀 Перейти на сайт](https://afrodit.pythonanywhere.com/)

## 📌 О проекте

Данный сайт был создан как веб-представительство игровой студии.  
Его основная задача — познакомить пользователей с командой, показать текущий проект в разработке и дать возможность следить за развитием игры.

На данный момент студия работает над своим первым проектом — психологическим хоррором в советском стиле, где игроку предстоит погрузиться в мрачную атмосферу и столкнуться с нарастающим чувством страха и преследования.

## ⚙️ Возможности

- главная страница с краткой информацией о студии
- описание текущей игры в разработке
- блок с информацией о команде
- адаптивный и современный интерфейс
- представление студии и проекта на сайте

## 🎯 Цель проекта

Сайт разработан для поддержки и продвижения начинающей игровой студии, а также для создания удобной платформы, где можно представить проект, рассказать о разработчиках и делиться прогрессом разработки.
